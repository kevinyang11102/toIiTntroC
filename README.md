# 家政管理系统 python492

## 项目概述

家政管理系统是一款基于Python+Django+MySQL开发的软件，旨在为家政服务行业提供全面、高效的信息管理解决方案。

## 技术栈

- 后端：Python、Django框架
- 数据库：MySQL
- 前端：HTML、CSS、JavaScript

## 功能模块

- 用户管理
  - 用户注册与登录
  - 用户信息编辑与查看
- 服务人员管理
  - 人员信息录入与编辑
  - 人员排班与调度
- 服务项目管理
  - 项目分类与设置
- 订单管理
  - 订单创建与跟踪
  - 订单状态更新
- 费用结算
  - 结算方式设置
  - 费用自动计算

## 系统角色

- 系统管理员
- 客户
- 服务人员
- 财务人员

## 运行环境

- Python 3.8+
- Django 3.2+
- MySQL 8.0+

## 部署步骤

1. 准备Python 3.8+环境
2. 安装Django 3.2+框架
3. 安装MySQL 8.0+数据库
4. 创建数据库及用户
5. 克隆项目代码
6. 修改配置文件中的数据库连接信息
7. 运行项目迁移命令，生成数据库表
8. 运行项目，启动服务

## 目录结构

```
家政管理系统/
│
├── apps/              # 应用目录
│   ├── user/          # 用户应用
│   ├── staff/         # 服务人员应用
│   ├── project/       # 项目应用
│   └── order/         # 订单应用
│
├── config/            # 配置文件目录
├── db/                # 数据库迁移文件目录
├── media/             # 媒体文件存储目录
├── static/            # 静态文件目录
└── templates/         # 模板文件目录
```
